# Moduł 4: Skille i pluginy

## Sekcja 1: Czym jest plugin?

Wyjaśnij uczestnikowi:

- Plugin to "dodatek" do Claude Code — rozszerza jego możliwości
- Analogia: "Plugin to jak rozszerzenie w przeglądarce Chrome. Chrome sam w sobie jest przeglądarką, ale z AdBlockiem, Grammarlyem czy LastPassem robi więcej. Plugin w Claude Code działa tak samo."
- Buddy-Claude, którego właśnie używacie — to plugin!
- Pluginy instaluje się z GitHuba (repozytoria kodu) jedną komendą
- Pluginy mogą zawierać: skille, konfiguracje, narzędzia

Zapytaj: "Widzicie? Już używacie pluginu — Buddy-Claude. Proste, prawda?"

## Sekcja 2: Czym jest skill?

Wyjaśnij uczestnikowi:

- Skill to konkretna umiejętność w ramach pluginu
- Plugin może mieć wiele skilli — jak aplikacja ma wiele funkcji
- Analogia: "Plugin to jak Canva. Skille to konkretne funkcje Canvy: tworzenie postów, prezentacji, infografik. Każdy skill robi jedną konkretną rzecz."
- W Buddy-Claude macie 4 skille:
  - `/tutorial` — ten przewodnik
  - `/explain` — słownik pojęć
  - `/exercise` — ćwiczenia
  - `/hint` — pomoc
- Skille uruchamiacie wpisując "/" i nazwę skilla

Poproś uczestnika: "Wpiszcie `/explain plugin` — zobaczcie jak skill explain wyjaśnia czym jest plugin. Skill wyjaśniający skille — meta!"

## Sekcja 3: Jak to działa "pod maską" (uproszczenie)

Wyjaśnij uczestnikowi (bardzo prosto):

- Gdy wpisujecie `/tutorial`, Claude Code znajduje odpowiedni plik z instrukcjami
- Ten plik mówi Claude JAK ma się zachowywać — co powiedzieć, o co zapytać, jak prowadzić
- Analogia: "To jak scenariusz dla aktora. Aktor (Claude) jest utalentowany, ale scenariusz (skill) mówi mu jaką rolę grać i co robić w danej scenie."
- Dlatego Claude odpowiada inaczej gdy używacie skilla vs. gdy piszecie normalnie

Zapytaj: "Czy to jest jasne? To ważne żebyście rozumieli — w przyszłości będziecie mieli więcej skilli dopasowanych do Waszej pracy."

## Sekcja 4: Co nadchodzi — zapowiedź

Wyjaśnij uczestnikowi:

- Buddy-Claude to dopiero początek
- W przyszłości pojawią się nowe pluginy i skille dedykowane dla Waszego zespołu
- Na przykład: asystent do analizy kampanii, generator treści w tonie marki, analizator ROI
- Te nowe skille będziecie instalować tak samo łatwo jak Buddy-Claude
- Wasza rola: nauczyć się teraz podstaw, żeby potem w pełni wykorzystać nowe narzędzia

Podsumuj moduł: "Teraz rozumiecie jak działają pluginy i skille. To fundamenty, na których zbudujemy Wasze przyszłe narzędzia AI. Chcesz przećwiczyć? Wpisz `/exercise 4`."
